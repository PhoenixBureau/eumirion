O stackinto:

Render target page's "joy" expression into the page as text. Especially, any Page Actions appearing in any text in the expression will take effect.

This is primary way let a page's "joy" expression affect the rendering of the content. Any command: buttons on a page will first modify its "joy" expression (here used as the stack) and then re-render the page. You can see and example of this on the See Animation in Action page.
