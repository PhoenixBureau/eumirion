O Using dip

Here's the process I went through to transform this (from "It's All Right" in Joy) into a version that has all the "arguments" to the left (on the stack) and the functions and combinators on the right (a program):

a sqr b sqr + c sqr =

Let's consider just this prefix:

a sqr b

In order to move sqr in front of b we can use the dip combinator. Its rule is:

... Q n == ... n [Q] dip

Apply it to our program prefix:

a b [sqr] dip

So far, so good. Let's add the rest of the program back in and see what we have:

a b [sqr] dip sqr + c sqr =

We can use that same transformation again to get:

a b c [[sqr] dip sqr +] dip sqr =

And we're done.
